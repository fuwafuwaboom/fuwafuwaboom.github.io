【Linux操作系统】Linux历史与环境搭建

发表于 2021-10-16 更新于 2021-11-04 分类于 【Linux操作系统】

本篇前言

Linux操作系统的入门篇，重点讲解Linux的发展历史，以及如何搭建Linux环境

学科历史与背景介绍

中国互联网时代背景

  • 1946年2月14日，第一款计算机“埃尼阿克”诞生，它是为了计算导弹弹道轨迹的而发明的。此时处于美苏冷战时期，两国都投入巨大资金研究高新技术，主导这件事的自然是美国军方。一般这种高科技，在完成其军用任务后，就会民用，发挥其经济效益，把之前投入研究的大量经费赚回来。
  • 于是，之后，计算机开始商业化。美国“梦工厂”硅谷，就是这样一个专门把高科技技术商业化的地方（商业化也就是从技术到产品）。
  • 1960年后，比尔盖茨、乔布斯，还有IBM公司，就开始利用这些技术，研究卖给用户的软硬件产品（电脑、服务器、操作系统OS operate system、还有基于OS的某种软件服务）
  • 当时虽然计算机技术开始商用，但是其保密程度仍然很高，直到90年代，苏联解体，冷战结束，美国的军用技术开始民用，有关计算机与互联网的技术封锁也开始减弱。
  • 94年的时候，我们国家政府开始介入，中国也顺利搭上互联网时代红利快车。
  • 中国第一批进入“互联网行业”的，其实并不是搞网络的，是做硬件的一批人，比如联想柳传志，华为任正非等。在计算机硬件技术兴起后，互联网应运而生，美国硅谷掀起了一波互联网创业潮，1900-2000十年间被硅谷称为互联网创业“黄金时代”。
  • 创业风气很快波及全球，尤其是中国的最先一批进军通信、计算机技术的人们，也开始了创业潮流。最早一批兴起的是搞门户网站的一批人，雅虎，新浪等。再后来，马化腾、李彦宏、马云、丁磊等一批“互联网风云人物”也在那时开始了各自的创业，马化腾创建QQ，马云创建互联网黄页，李彦宏模仿谷歌做搜索引擎。中国互联网巅峰10年（2000-2010）拉开帷幕，开始了十年的群雄逐鹿，直到2010年基本形成互联网行业B（百度）A（阿里）T（腾讯）三足鼎立，游戏邻域网易腾讯平分秋色，手机行业小米、华为、vivo、oppo、魅族等国产手机迅猛发展。
  • 2008-2012年后手机加速普及，雷军的小米让人人让手机价格骤减，人人都能够拥有自己的移动设备，世界基本进入移动互联网时代。衣食住行娱、移动支付迅速与移动互联网结合，涌现出一批如美团、京东、拼多多、哔哩哔哩的公司。
  • 直到2019年字节跳动异军突起，华为5G技术突破奠定通信一哥。可以说中国21世纪以来20年的互联网发展与美国齐头并进。

Unix→Linux

  • 美苏冷战结束后，大批军用高新技术开始民用，其中之一就是操作系统。最早的进入企业级商用的操作系统是unix，只有大公司和常春藤联盟的高校可以使用。由于操作系统研发投入巨大，所以最早的unix操作系统是付费使用的
  • 最早版本的unix操作系统的是由肯·汤普森书写，由于当时没有C语言，所以肯是用汇编语言写的。由于汇编语言和硬件关系很大，可移植性很差，丹尼斯·里奇发明C语言后，一人认为自己是所有操作系统的爸爸，一人认为自己是所有编程语言的爷爷，所以两人情投意合，合作用C语言写成了新版本的unix操作系统。C语言的很多语句是跨平台性的，大大增加了程序的可移植性
  • 由于unix操作系统在常春藤高校中盛行，高校中的大神，雷纳斯·托瓦兹也用到了unix操作系统，但是由于unix是收费的，使用用户过少，而且unix至今并未开源，源代码未公开，所以托瓦兹自己根据unix的功能写了一款操作系统。这款操作系统相对独立，作者将其命名为linux操作系统，Linux是Linux not unix的缩写，表现了作者对这种闭源收费行为的反抗。接着他做了一个创世纪的事情，他将这款操作系统免费开源了
  • 最新的linux官网为kernel.org,至今我们仍然可以从中下载源代码，甚至对其发展做出贡献

开源精神

  • 开源指的是将自己项目的源代码毫无保留的公开，共享知识
  • 开源后，一定会有很多人查看我们的开源项目，群众的眼睛是雪亮的，他们会帮助我们指出问题，修正代码
  • 开源后，全世界人民都会盯着我们的代码，而且基本不会有人瞎搞，因为一旦有人瞎搞并上传，全世界人民都会看到你的“恶行”，你的个人履历上会留下不可磨灭的一笔。而且参与开源项目的人基本都是使命驱动的。所以开源的项目一定是更稳定，更高效的。
  • 所以，“源码之前，了无秘密”
  • 那为什么会有人无偿参与这些开源项目呢？首先，这批参与Linux项目的人，一定是行业的翘楚（民间技术大神、顶级黑客、各大公司的前沿研发人员），而行业中的翘楚一定是不缺钱的。人一不缺钱，就开始求名望。所以这些顶尖的人才，是基于使命感驱动，希望巩固自己行业地位而参加的。而这样的开源代码还有一大批不太顶级的人盯着。所以这样的项目是没有理由不成长飞速的。

Windows/Mac & Linux

  • 在用户端，Windows基本处于垄断地位，而企业端，基本是linux处于垄断地位。
  • 为什么linux会在企业端垄断呢？原因在于：企业是需要一周7*24不停的响应客户端的，而linux是最稳定而且免费的开源操作系统，不会有天价的收费，不会有受制于人的版权隐患，不会有信息安全和系统不稳定的隐患
  • 熟练运用linux后，我们基本是不屑于使用windows操作系统的。linux是能够保证开机几年的时间不关机仍能高效运行的，而windows会不断积攒后台垃圾，速度越来越慢。所以总体来说，linux的代码质量是高于windows一个层级的。
  • 因为linux是开源的，所以这些顶级工程师是不会将这款操作系统商业化的，所以他们追求的一定是系统的性能，而不会讨好个人（付费端用户）的，所以linux的操作比较复杂（我们要追求达到纯命令行操作的地步）
  • 所以虽然windows相对不稳定，它仍然在用户端形成了垄断。最早期的操作系统，不管是windows还是mac os基本都是命令行的。直到一家从事打印机业务的公司——施乐，在它的顶尖实验室中研发了第一款图形界面的os，乔布斯听说了以后，以高价买取了一次参观内部技术的机会，后来他带着一批顶尖工程师，前往参观。参观归来，乔布斯火速成立研发小组，研发出了苹果的第一款图形化操作系统——麦塔金。这种图形化的操作系统对命令行操作系统在用户体验上的优势是碾压式的。
  • 此时微软CEO比尔盖茨就着急了，他仗着自己财大气粗，投入大量人力和财力，研发了windows NT、windows XP等图形化操作系统。从此，乔布斯就和比尔盖茨彼此看不顺眼。而让乔布斯和比尔盖茨更看不爽的是Linux之父雷纳斯托瓦兹，因为他的开源行为无疑是抢夺了windows和mac os的饭碗
  • 苹果是一家软硬件齐头并进的公司，所以他们的产品研发成本很高，价格就相对昂贵，比尔盖茨发现自己在硬件领域落后苹果，就拉上了另一家公司IBM，希望一起对抗苹果，但IBM格局太小，它不敢设想操作系统这样的“高科技”会走入寻常百姓家，所以微软就和另一家公司Intel合作，从此微软做软件，Intel做硬件，又和松下、日立、三星等公司合作，所以成本降低，逐步抢占了pc端的市场份额。这也是为什么windows会处于垄断地位

Linux的主流应用场景

1.企业后台

  • 由于现在的大公司的用户体量都是上亿级的，需要大量的土地建楼，盖机房，搭载操作系统，成本巨大，只有少数头部公司能够使用
  • 而很多中小型公司的无法负载成本的，所以大公司就开发了自己的云服务系统，以租赁的形式，使用“云机房、云机器”，这种to B的方式实际上也可以让我们个人使用

2.智能设备

  • 电视、手机、智能手表
  • 美国一家手机公司在Linux内核的基础上，开发了手机端的操作系统
  • 谷歌买下了这家公司，然后将这款操作系统免费开源，命名为Android，也就是安卓操作系统。为什么要花重金开源呢？因为当时乔布斯的Apple如火如荼，在全球手机端越来越火，所以谷歌开源安卓的本质是为了对抗苹果

3.嵌入式领域

  • Linux系统的轻载性使其可以在足够小的硬件上运行
  • 嵌入式系统技术的发展，使得一大批机械电子设备开始智能化时代

4.机器人操作系统

  • 2007年，ROS—robot operating system，世界第一款机器人操作系统发明于斯坦福机器人实验室，创始人表达了希望创造出机器人Linux的愿景，随后开源了整个项目
  • 2011年，ROS Answers出现，ros进展飞速，随后第一款基于ros的商业机器人研发成功
  • 2018年，ROS引起了微软和亚马逊两大巨头的注意，微软将ros核心部分移植到windows,亚马逊基于ROS开发了自己的RoboMaker
  • 至今，ROS2的研发仍在如火如荼的进行

5.物联网

  • 物联网操作系统，目前以华为的鸿蒙Harmony OS为代表
  • 华为在开放鸿蒙后开源了整个项目，但由于美国的技术封锁，目前只有中国技术人员在如火如荼的开发中

操作系统的版本

Linux : Linux内核（开源） + 命令行交互

windows : windows内核（闭源） + 图形界面交互

mac : mac内核（闭源） + 图形界面交互

许多个人、组织、私人公司会基于开源内核（Linux）进行二次开发（比如Android）

还有许多带有图形界面的商业化发行版：Ubuntu、RedHat、debian、kali等等

但值得注意的是，windows7,8,10,11是内核发行版，是纯技术的

所以我们对于Linux版本的认识要有两个部分：内核发行版和界面发行版

Linux环境的搭建与使用

Linux环境的搭建

1.物理系统

直接将自己的电脑设备的操作系统重装成Linux系统

不推荐小白使用，因为极有可能因为操作不当引起电脑的报废

推荐有一定技术力，而且追求Linux纯命令行操作的“极客”使用

b站有教程

2.双系统

将电脑的软硬盘分出一部分建立双系统

安装好后使用相对较容易，开机后选择windows/linux即可

安装过程较繁琐，教程网上都可以找到

3.虚拟机

直接在windows下打开linux内核的虚拟机

推荐VMware

推荐小白使用，容错率高

4.云服务器

推荐富有小白使用，操作简单

国内的阿里云、腾讯云、华为云都可以使用，费用100-200/年不等，学生凭借学生证或身份证有优惠，由于我们刚入门，哪个便宜买哪个

云服务器下登录Linux

1.Xshell登录

下载Xshell学生和家庭版本，直接连接自己购买的云服务器公网，输入密码登录

对于上面的warning警告，不需要在意，这是由于我们所采用的终端和linux系统不是100%兼容导致的，所有采用此方法的人都会出现

在Xshell中输入

ssh root@公网地址

再输入登录密码也可以直接登录

Xshell的复制粘贴：

复制 ctrl+insert

粘贴 shift+insert

2.win命令提示符

ssh 回车

ssh 用户名@公网地址

输入密码（linux输入密码是不显示的）

登录成功

但这种方法不太推荐是因为很可能会有各种bug出现

云服务器下新建账号

登录root

whoami

ls /home

adduser 新用户名

passwd 新用户名

输入密码

确认密码

这样就不需要登录root主用户了，可以登录许多用户

云服务器下删除已有账号

登录root

userdel -r 用户名
