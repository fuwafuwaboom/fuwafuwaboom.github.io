链式调用与嵌套调用

链式调用：用一个函数的返回值作为另一个函数的参数

def isOdd(num):
    if num % 2 == 0:
        return False
    return True
def add(x, y):
    return x + y
print(isOdd(add(5, 5))) # 注意太多层会影响可读性

嵌套调用：函数中调用另一个函数，层次可以任意层

def a():
    print('a')
    
def b():
    print('b')
    a()
    
def c():
    print('c')
    b()
    
def d():
    print('d')
    c()
d()

局部变量与函数栈帧

函数栈帧：调试器的左下角可以看到函数之间的调用栈，每一层调用关系叫做函数的栈帧，每个函数的局部变量就是在栈帧中体现的

函数结束，对应的栈帧消亡，内部的局部变量也消亡

所以不同作用域的局部变量理论上是可以同名的，但是为了增加可读性，尽量不要出现同名的变量

函数递归

函数调用自己就是递归

n!：

def factor(n):
    if n == 1:
        return 1
    return n * factor(n - 1)

注意的是不同的调用是不同的栈帧，栈帧中的局部变量也不相同

递归的代码一定不会无限执行，而是不断逼近递归结束的条件

递归代码写法很简单，但是执行过程很难

所以理解递归代码需要利用画图+调试器辅助分析

递归需要递归结束条件+递归公式两个基本要素

递归也容易造成栈溢出（写错了无限递归）

一般递归版本都可以转化成等价的循环代码，而且循环写法运行更快速

递归的优点就是可以解决本身就是通过递归定义的问题：比如二叉树，斐波拉契数列等等

函数形参的默认值

上述函数可以写成

def add(x, y,d debug):

但是每次都要写debug的值非常麻烦，可以设置默认值：

def add(x, y,d debug = Flase):
    if debug:
        print(f'x = {x}, y = {y}')   
    return x + y

print(result)

列表和元组

变量就是存储空间，用来表示数据

如果表示的数据少，直接定义几个变量就可以了

num1, num2, num3 …

但是如果表示的数据太多，就不能用这种方式定义了

Python中的列表和元组就是用一个变量表示许多数据（类似于C/C++的数组）

列表和元组大部分的功能都是差不多的，唯一的区别是：列表是可变的（创建好了之后随时能改），元组是不可变的（创建好了之后改不了，要改，只能丢弃旧的，搞个新的）

列表的切片操作

切片就是一次取出一组连续的元素，相当于取出一段子列表

# 1.切片操作的基本使用
a = [1, 2, 3, 4]
print(a[1:3])# 取到下标为1一直到下标为3的元素，且包含1不包含3，也就是说切片操作是前闭后开的

# 2.切片操作可以省略边界
print(a[1:]) #省略后边界意思是从开始位置一直取到整个列表结束
print(a[:]) #取出列表所有元素，得到列表自身

# 3.切片操作还可以指定步长
a = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
print(a[::1]) #1指的是步长
print(a[::2]) #结果是[1,3,5,7,9]
print(a[::3]) #结果是[1,4,7,0]
print(a[1:-1:2]) #结果是[2,4,6,8]

# 4.步长的数值还可以是负数，意思为从后往前取元素
print(a[::-1]) #结果是[0,9,8,7,6,5,4,3,2,1]

# 5.当切片的范围超出有效下标时并不会异常，而是尽可能的把符合要求的元素获取到
a = [1, 2, 3, 4, 5, 6, 7, 8, 9, 0]
print(a[1:100]) #结果是a = [1,2,3,4,5,6,7,8,9,0]

列表的遍历操作

遍历：把每个元素依次取出并进行某种操作（不仅限于列表）

遍历要求不重不漏

遍历往往要搭配循环

# 1.使用for循环遍历列表
a = [1, 2, 3, 4, 5]
for elem in a:
    print(elem)
    
# 2.使用for循环+下标
a = [1, 2, 3, 4, 5]

a = [1, 2, 3, 4, 5]

字典

字典的创建

字典表示一种存储键值对的数据结构

键值对：计算机中一个非常重要的概念

键（key）

值（value）

键值对就是一种映射关系，根据key可以快速找到value

比如学校中的每个同学的学号，可以根据学号找到同学

在Python的字典中，可以同时包含许多键值对，但是要求这些键不能重复，换句话说，一次只会操作一个key

# 1.创建字典
a = {} #大括号表示字典
b = dict() #dict = dictionary

# 2.创建字典的同时设置初始值
a = {'id':1, 'name':'zhangsan'} #大括号里写键值对，用“:”表示键值对应，用“,”分割键值对
# 字典中key的类型和value的类型不必都一样
# 字典对于key的类型有约束，但是对value的类型没有约束

# 3.一般建议按照下列多行形式书写字典
a = {
    'id':1, 
    'name':'zhangsan',
}

字典查找Key

先看看key是否在字典内存在

# 1.使用in判定
a = {
    'id':1, 
    'name':'zhangsan',
}
print('id' in a) #判定id是否存在，是返回True，不是返回False
# 注意in只能用来判定key是否存在，不能用来判定value
print('id' not in a) #判定id是否不存在

# 2.使用[]来根据key获取到value
a = {
    'id':1, 
    'name':'zhangsan',
    100:'list'
}
print(a['id']) #结果就是'id'对应的value，也就是1
print(a[100]) 
# 注意key也可以是整数，这时a[100]就不是list而是取value操作
print(a['class']) #这是一个读操作
# 如果查找的key不存在，就会报错“KeyError”
# 与列表想比，字典的查找中in更高效，而list使用[]更高效，因为字典背后使用的数据结构是哈希表，哈希表很适合遍历

字典的新增修改删除

# 1.在字典中新增元素
a = {
    'id' = 1,
    'name' = 'zhangsan'
}
a['score'] = 90 #字典的新增 这是一个写操作

# 2.在字典中根据key修改value
a['score'] = 100 #key存在就是修改 key不存在就是新增 和变量创建和修改类似

# 3.使用pop方法根据key来删除键值对
a.pop(name)
print(a)

# 注意字典的各种操作是操作key的 所以key至关重要

字典的遍历

遍历就是把一个可迭代对象的元素依次取出来并进行一些操作，整个过程要求不重不漏

字典设计的初衷并不是为了遍历而是增删改查，所以字典的遍历效率会差一点（哈希表能够以常数级的时间复杂度来实现增删改查，也就是说不论字典中有多少元素，实现增删改查的操作都是固定时间）

# 直接使用for循环遍历字典
a = {
    'id':1,
    'name':'zhangsan',
    'score':90,
}

for key in a:
    print(key, a[key])
    
# 事实上，哈希表的存储顺序是无序的，但是Python中作了特殊处理，能够保证遍历出的顺序和插入的顺序一致
# 所以Python的字典，结合了哈希表和队列的结构特点

# 其他遍历方法
print(a.keys)
# 返回了dict_keys([])自定义类型 实际上和列表非常相似
print(a.values)
# 返回了dict_values自定义类型
print(a.items)
# 得到的是所有的键值对 返回了dict_items自定义类型 首先是一个列表 列表的每个元素是一个元组 元组包含了键和值
# dict_items([('name','zhangsan')])

#上面的语句可以写成下列形式
for key,value in a.items():
    print(key, value)

可哈希的类型

字典中对key是有要求的，什么样的类型能作为字典的key呢？

可哈希的类型才能作为字典的key

# 使用hash函数能够计算出一个变量的哈希值
print(hash(0))
print(hash(3.14))
print(hash('hello'))
print(hash(True))
print(hash((1, 2, 3)))
# 整数 浮点数 字符串 布尔值 元组 都可以计算哈希值

print(hash([1, 2, 3])) #报错 因为列表是不可哈希的(unhashable)
print(hash({})) #报错 因为字典是不可哈希的

我们可以简单的认为，一个不可变的对象一般就是可哈希的，一个可变的对象一般就是不可哈希的。而列表和字典都可以进行增删改查，是可变的对象，是不可哈希的

字典、列表、元组 都是Python中非常常用的内置类型

它们都可以包含其他元素，所以又叫它们“容器”或者“集合类”
