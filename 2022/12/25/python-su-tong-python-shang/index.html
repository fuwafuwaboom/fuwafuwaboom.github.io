Python的优缺点

优点：

  • 语法简洁易上手
  • 功能强大，用途广泛
  • 生态丰富，海量第三方库
  • 与C/C++相互调用很容易

缺点：

  • 执行效率慢
  • 对多核心并发编程支持偏弱
  • 变量是动态类型，对大型项目和多人合作项目并不友好

基础语法

常量和表达式

print(1 + 2 * 3)

先乘除后加减

print(1 + 2 / 3)

结果是1.66666665，因为IEEE标准的浮点数有些许误差

1 + 2 + 3称为表达式

表达式的运算结果，称为表达式的返回值

1 2 3这样的数字，称为 字面值常量

+-*/称为 运算符/操作符

变量和类型

给定四个分数，求他们的方差

方差：每个数字与平均值的差值平方相加除以项数-1

avg = (67.5 + 89.0 + 12.9 + 32.2) / 4
total = (67.5 - avg) ** 2 + (89.0 - avg) **2 + (12.9 - avg) ** 2 + (32.2 - avg) ** 2
print(total / 3)

变量可以视为是一块用来存储数据的空间（一个纸盒子）

= 赋值运算符

变量名必须是 数字/字母/下划线 且 数字不能开头 且 不能与Python的关键字重合 且 大小写敏感

变量名尽量是 描述性单词 且 可以使用多个单词描述(PersonInfo/person_info)

变量使用：

a = 10 #初始化
b = a 
print(b)

a = 20 #赋值
print(a)

变量类型：

  1. 整数 int
a = 10
print(type(a))

结果：<class ‘int’>

Python中的变量类型可以根据初始化直接定义

Python中int可以表示的数据范围是无穷的 （所以没有long short 类型）

  1. 小数 float

Python中的浮点数就是双精度浮点数

Python的设计哲学：解决一个问题只使用一种方案

  1. 字符串 str

Python中要求用引号把一系列字符引起来，单引号双引号均可

单双引号可以进行’I love the book “freedom” ‘ (pycharm绿色波浪线是检查拼写错误)

Python还有三引号 ‘’’，如 ‘’’ My ‘name’ is “freedom” ‘’’

len()函数可以打印字符串的长度，打印字符串有多少个字符

字符串可以拼接：a1 + a2 就是把a1a2两个字符串拼接（所以不能把字符串和数字混合相加）

Python有两类错误：语法错误（编写代码时就发现）和 运行错误（运行后发现的错误）

  1. 布尔 bool

布尔类型只有两个值

真：True

假：False

（首字母大写！）

布尔值用于逻辑判定

数据类型的意义：

  1. 不同的类型表示占用不同的空间（int默认4字节可动态扩容/float 8/bool 1/str 变长）

  2. 不同的类型，对应的操作权限是不同的

动态类型特性：

程序运行过程中，变量的类型可能会发生改变

a = 10
print(type(a))

a = 'hello'
print(type(a))

a = True
print(type(a))

class<’int’>

class<’str’>

class<’bool’>

（C/C++是静态类型）

a:int = 10 #可以进行变量类型的声明 但不会改变其动态类型特性 只是锦上添花

输入和输出

最简单的办法就是使用控制台：

用户能够输入命令，向计算机安排工作，并且计算机可以显示出程序的运行结果

这样的程序就叫做控制台

日常生活中最常见的是图形化界面，可以大大降低用户的使用门槛，但是图形化界面的程序编写起来却不太容易

格式化字符串 f - string：

print(f"a = {a}")

输入函数input：

num = input('请输入一个整数：')
print(f'您输入的整数是{num}')

input返回的值是一个字符串

当需要使用用户输入的值时，需要类型转换

a = input('请输入第一个整数：')
b = input('请输入第二个整数：')

a = int(a)
b = int(b)

print(f'a + b = {a + b}')

运算符

运算符名称 运算符
算术运算符 ** * / + - // %
关系运算符 < <= >= > == !=
赋值运算符

** 乘方 可以支持整数次方 也可以支持小数次方

/ 除以0或者0.0都是出错的

% 是取余数

// 是地板除法/取整除法，会对结果进行向下取整（选择最小的整数，所以负数取整方向是反的）

关系运算符的表达式值是布尔值（True/False）

关系运算也可以比较字符串（字典序，字符串在字典上越靠前越小）

浮点数进行关系运算有风险：0.1 + 0.2 不等于 0.3 正确的方法是做差然后看结果是否小于误差范围

python支持连续比较

print(-0.000001 < (a - b) < 0.000001) # 正确比较浮点数的写法

条件与循环

条件语句

#include//iostream是库头文件，不加.h，但是我们引用自己定义的头文件需要加.h
using namespace std;
int main()
{
    int n = 0;
    cin >> n;// >> 流提取运算符/输入运算符
    cout << n << endl;// << 流插入运算符/输出运算符
    return 0;
}

多次输入以空格和换行间隔

endl等价于输出了一个换行符'\n'

输出是从左向右输出

至于cin和cout到底是什么，在学习了类和对象以后再做解释

循环语句

C++的输入输出可以自动识别类型，这是比C语言方便的

#include
using namespace std;
int main()
{
    char ch = 'A';
    int i = 10;
    int* p = &i;
    double d = 3.1415926;
    cout << ch << endl;
    cout << i << endl;
    cout << p<< endl;
    cout << d << endl;
    return 0;
}

C++的小数默认只输出小数点后5位，不太方便，此时推荐配合printf使用

printf（“%.2f",a);表示小数点后两位

其实对于连续输出同类型的，比如结构体，使用C语言反而更方便，因为不用多写那么多<<符号

所以建议在C++中不用纠结于到底用哪一个，而是哪个好用用哪个

函数

相对C语言，C++增加了缺省参数的概念用于补充函数的定义

在定义函数的时候直接指定形参的值，就是缺省参数

缺省参数仅在没有传参的时候起作用，一旦传参就没这个参数什么用了
