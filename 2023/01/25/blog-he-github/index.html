blog和github

Posted on 2023-01-26 Edited on 2023-01-27 In 【编程基本功】

本篇前言

记录一下怎么上传博客和GitHub

Github

创建仓库连接到本地

1.点击New

2.设置仓库参数 创建仓库

3.与本地仓库相链接

右击想要创建的文件夹的空白处，点击Git Clone

把刚刚的网址复制进去，然后点击ok

这样我们的本地仓库也创建好了

在本地仓库加入文件就可以进行上传了

上传文件

1.add

2.commit

输入一些描述性文字

3.push

点击push

成功

然后在github上就可以看到上传的文件了

上传博客hexo

到blog根目录shift右击空白处，点击powershell

hexo g

hexo s （之后用ctrl+c退出）

hexo d

上传完毕
