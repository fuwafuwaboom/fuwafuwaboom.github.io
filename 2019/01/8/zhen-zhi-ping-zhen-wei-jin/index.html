针织秘诀

理解原理 细节把控

认真模仿 用力适中

熟能生巧 大巧不工

平针围巾基本步骤

1.计算每排针数:

int max = 10*2 + 1;

2.起针

起max针

3.针织开始

第一排：正挑不织→正针→反针→正针→反针→…→反针 正针起，反针结

第二排：正挑不织→反针（记住反针第一针的时候，正挑不织后要先逆时针绕线再起针）→正针→反针→正针→…→正针 反针起，正针结

…（奇数排一正一反 偶数排一反一正）

4.结针

结max针

起针

int n = 0;

while(n=20)

{

n++;

}

正挑不织

抽针

第一针直接挑出来，不绕线

正针

1.正针前顺时针绕线（第一针不用）

2.正孔插进去

3.逆时针绕线

4.挑出来

5.松掉

反针

1.反针前逆时针绕线

2.反孔插进去

3.逆时针绕线

4.挑出来

5.松掉

结针

至此，围巾步骤结束
